# *Back-End* (Server)

Created by [SSbit01](https://ssbit01.github.io) with [Socket.IO](https://socket.io/) and [TypeScript](https://www.typescriptlang.org/). Remember to install local packages.

You can run the following *npm scripts*:

- `dev`: starts the project in development mode.
- `build-only`: compiles the project, the generated file will be located in `../dist` as `index.js`.
